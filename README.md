# Assignment-1

Structure and construction of digital platforms (HTML, CSS, JavaScript connections)





Filer och deras roll:

index.html → Skapar struktur (rubrik, text, knappar, main, footer)

style.css → Bestämmer utseende (färger, typsnitt, layout, hover-effekter)

app.js → Lägger till interaktivitet och dynamiska element via DOM







Hur filerna samverkar:

Webbläsaren laddar HTML → DOM skapas.

CSS appliceras → sidan får utseende.

JS körs → knappar och dynamiska element aktiveras.

När användaren klickar på en knapp ändras DOM → webbläsaren uppdaterar vyn direkt.




Interaktivitet:

Ändra text= Klick ändrar #intro-text.

Återställ text= Klick återställer ursprungstexten.

Visa tid= Klick visar aktuell tid i #timeDisplay.

En lista byggs dynamiskt med JavaScript: varje frukt i arrayen blir ett li i en ul.


Sammanfattning:

HTML: Struktur

CSS: Utseende

JS: Interaktivitet




Kombinationen bildar en enkel digital plattform där användaren kan interagera utan att sidan laddas om.
